# 基于Linux的多用户航班购票系统

## 服务端

1. 服务端流程

    - main函数启动
    - 增加一个分离属性线程加载用户数据
    - 增加一个分离属性线程加载航班信息数据
    - 来一个访问就增加一个线程来保持连接是阻塞io

## 客户端

1. 首页登录

    ```c
    if 用户已注册
            登录界面
    else 用户未注册
            注册页面

    ```

2. 登录界面

    - 登录
    - 注册
    - 忘记密码
    - 返回

3. 注册页面

    - 姓名
    - 手机号
    - 手机验证码

### 用户登录成功界面

1. 航班列表

    - 获取列表 读取每一条消息,消息格式比如成功 200:20:数据

2. 查询航班

3. 快速查询

4. 订购航班

5. 已购航班

6. 退票改签

7. 查询余额

8. 充值账户

9. 修改信息

10. 取票服务

11. 打印发票

12. 聊天功能

13. 朋友圈

14. 退出登录
