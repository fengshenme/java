功能点分析
=======================
   (1)把C语言写的读取卡号代码融入QT工程
         思路：参考摄像头V4L2代码融入QT工程(一模一样的思路)
   (2)如何判断第一次刷卡还是第二次刷卡
         思路一：刷卡成功之后，调用select语句去查询表格中是否有这个卡号
                有卡号 --》说明现在是第二次刷卡，出车
                没有卡号 --》说明现在是第一次刷卡，入车
                温馨提醒：不能用exec()的返回值去判断查询是否成功
         思路二：卡号字段修饰成unique(唯一性)
                 刷卡成功，直接插入卡号到表格中
                 插入成功 --》说明现在是第一次刷卡，入车
                 插入不成功 --》说明现在是第二次刷卡，出车
